Name Alarm Display
Number GST.52.5
Description

All alarms received from GST managed network elements shall be displayed showing: Network element identified, Network element sub-system identified, Occurrence time and date, Alarm criticality and severity level, Description, Probable cause, Alarm state (pending, acknowledged or terminated)
